Posted on 周日 06 十一月 2016 in python • Tagged with jupyter, virtualenv

首先创建独立的运行环境

mkvirtualenv scrapy

在这个环境中安装jupyter和ipykernel

pip install jupyter
pip install ipykernel

jupyter --path 查询jupyter的路径

    C:\Users\your_user_name\.jupyter
    c:\users\your_user_name\envs\scrapy\etc\jupyter
    C:\ProgramData\jupyter

    C:\Users\your_user_name\AppData\Roaming\jupyter
    c:\users\your_user_name\envs\scrapy\share\jupyter
    C:\ProgramData\jupyter
