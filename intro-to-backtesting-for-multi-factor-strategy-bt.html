Intro to Backtesting Framework for Python - bt¶

  • 项目地址
  • 文档
  • bt简介: bt是一个量化交易测回测框架，基于Python。 这个框架使用户能够方便的创建基于不同Alogs的回测策略。 用户可以用bt创造灵活的、可测试的、可重用的策略块。 bt的目标：免去quant重新造轮子的时间，使他们能够专注于策略的开发。bt的作者，也是ffn的作者。bt是目前我看到的唯一一个可以用于因子测试的开源项目。

# 创建策略
s = bt.Strategy('s1', [bt.algos.RunMonthly(), #每月调仓
                       bt.algos.SelectAll(),  #选择所有股票
                       bt.algos.WeighEqually(), #等权重构建组合
                       bt.algos.Rebalance()]) #执行调仓
In [9]:
# 创建回测并执行
test = bt.Backtest(s, data)
res = bt.run(test)

# 下面这个策略不再使用简单的等权重方法，而是基于股票的波动率确定持仓的大小
# 如果波动率高，那么持仓就低些，反之，如果波动率底，就提高持仓

# 创建策略
s2 = bt.Strategy('s2', [bt.algos.RunWeekly(),
                        bt.algos.SelectAll(),
                        bt.algos.WeighInvVol(), # <- 关键！
                        bt.algos.Rebalance()])

# now let's test it with the same data set. We will also compare it with our first backtest.
test2 = bt.Backtest(s2, data)
res2 = bt.run(test, test2)
