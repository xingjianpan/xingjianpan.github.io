针对每个Virtualenv，创建独立的Jupyter Kernel (Windows 10)

Posted on 周日 06 十一月 2016 in python

首先创建独立的运行环境

mkvirtualenv scrapy

在这个环境中安装jupyter和ipykernel

pip install jupyter
pip install ipykernel

jupyter --path 查询jupyter的路径

config:
    C:\Users\your_user_name\.jupyter
    c:\users\your_user_name\envs\scrapy\etc\jupyter
    C:\ProgramData\jupyter
data:
    C:\Users\your_user_name\AppData\Roaming\jupyter
    c:\users\your_user_name\envs\scrapy\share\jupyter
    C:\ProgramData\jupyter
runtime:
    C:\Users\your_user_name\AppData\Roaming\jupyter\runtime

找到runtime对应的路径，进入他的父目录。创建一个叫做kernels的文件夹

C:\Users\your_user_name\AppData\Roaming\jupyter\kernels

在kernels文件夹内创建scrapy文件夹，在scrapy文件里创建名为kernel.json的配置文件

C:\Users\your_user_name\AppData\Roaming\jupyter\scrapy\kernel.json

kernel.json内容如下

{
 "argv": [ "C:\\path\\to\\your\\virtualenv\\folder\\python", "-m", "ipykernel",
          "-f", "{connection_file}"],
 "display_name": "scrapy",
 "language": "python"
}

如果一切顺利，那么名为scarpy的kernel将会出现在jupyter notebook中。

jupyter virtualenv
